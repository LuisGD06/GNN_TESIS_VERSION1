# Selección de algoritmos

## Baselines clásicos

- Logistic Regression.
- Random Forest.
- XGBoost o LightGBM.

## GNN para nodos

- GCN como baseline GNN.
- GraphSAGE como modelo principal por su enfoque inductivo.
- GAT como modelo comparativo por atención sobre vecinos.

## Subgrafos

- Features estructurales + Random Forest/XGBoost.
- GIN + global_mean_pool.
- GraphSAGE + pooling.

## Mejoras futuras

- Deep Graph Infomax.
- GLASS-style labeling.
- Modelos temporales.
- Grafos heterogéneos con Elliptic++.
