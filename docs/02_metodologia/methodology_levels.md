# Niveles metodológicos

## Nivel 1: Análisis exploratorio del grafo

Carga, validación, estructura temporal, distribución de clases, nodos, aristas y conectividad.

## Nivel 2: Baselines clásicos

Modelos tabulares: Logistic Regression, Random Forest, XGBoost o LightGBM.

## Nivel 3: Clasificación de nodos con GNN

Modelos GCN, GraphSAGE y GAT.

## Nivel 4: Análisis de subgrafos

Extracción de vecindarios k-hop, caminos, fan-in, fan-out, cadenas y ventanas temporales.

## Nivel 5: Clasificación de subgrafos

Modelos clásicos sobre features de subgrafos y modelos GNN con pooling global.

## Nivel 6: Exportación a plataforma analítica

Generación de predicciones, scores de riesgo, alertas, métricas y archivos para dashboard.
